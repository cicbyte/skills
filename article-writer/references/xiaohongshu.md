## 平台定位

小红书是**视觉驱动的种草社区**，用户以年轻群体为主。技术内容以「学习笔记」「工具推荐」「效率提升」为切入点。文字部分限制 1000 字，必须精炼。瀑布流推荐机制，封面图决定点击率。

## 标题风格

- **数字 + 痛点 + emoji** 是黄金公式
- 标题控制在 20 字以内
- 常见句式：
  - 「5个Python技巧，效率直接翻倍🔥」
  - 「程序员工具分享｜这个插件太好用了」
  - 「转码成功！我是这样学的💻」
  - 「后悔没早点知道的VS Code快捷键」
- 适当使用感叹号增强情绪
- 标题中可加 1-2 个 emoji

## 笔记结构

### 标准结构（< 1000 字）

```
封面图（精心设计，决定生死）

标题（吸睛 + emoji）

开头（2-3行）
  → 点明痛点/分享动机
  → 引发共鸣：「每次xxx都觉得好烦？」

正文（分点列出）
  → 1️⃣ 2️⃣ 3️⃣ 逐步讲解
  → 每个要点 2-3 行
  → 配图/截图穿插

结尾（1-2行）
  → 互动引导
  → 话题标签 #xxx #yyy
```

## 语言风格

- **最口语化、最亲切**的平台
- 像和朋友聊天：「姐妹们/兄弟们」「真的绝了」「好用哭了」
- 即使是技术内容，也要用大白话
- 适当使用小红书热词但不过度
- 强调**个人体验感**和**真实感受**
- 第一人称叙述：「我发现」「我一直在用」「亲测有效」

## 排版规范

### Emoji 使用（核心要素）
- 大量使用 emoji 作为视觉分隔和情绪表达
- 推荐高频 emoji：🎯 💡 🔥 🚀 ✅ ⚡ 💻 📌 🛠️ 👆 👇 🌟 💪 🎉 😭 🤯
- 每段开头用一个 emoji 标记类型
- 用 emoji 替代传统标点做分隔

### 段落规范
- 每段 **1-2 句话**，不超过 3 行
- 大量换行，留白充足
- 用 ①②③ 或 1️⃣2️⃣3️⃣ 列要点
- 用 ✅❌ 做正反对比
- 重点内容前后加 emoji 突出

### 图片规范
- **封面图是最重要的元素**，决定是否被点击
- 封面图风格：代码截图 + 大字标题
- 内页图：步骤截图、效果对比图
- 推荐尺寸：3:4 竖版（1242×1660px）
- 图文配合：一张图配一段文字

## 技术元素处理

### 代码
- ❌ **不贴代码**（小红书不支持代码块，贴了也看不了）
- ✅ 改用「代码截图」：用 Carbon 或类似工具生成美化截图
- ✅ 关键代码用伪代码或自然语言描述
- ✅ 聚焦「思路」而非「实现」
- 示例转换：
  ```
  原文（❌）：for i := 0; i < len(arr); i++ { ... }
  小红书（✅）：用 for 循环遍历数组，对每个元素做判断就行啦～
  ```

### 仓库链接
- ❌ 不直接贴链接（小红书限制外链）
- ✅ 「GitHub 搜索 🔍 用户名/仓库名」
- ✅ 引导至评论区/私信获取链接
- ✅ 用截图展示仓库 README 和 Star 数

### 工具/资源推荐
- 用表格截图或卡片图展示
- 标注「免费/付费」「推荐指数 ⭐⭐⭐⭐⭐」
- 配合使用前后对比

## 爆款技巧

1. **选题聚焦**：一篇笔记只讲一个点，不贪多
2. **痛点共鸣**：让读者产生「这说的就是我」的感觉
3. **情绪触点**：适当表达惊讶/后悔/兴奋的情绪
4. **可复制性强**：给出可直接操作的步骤
5. **评论区运营**：主动回复，引导讨论
6. **话题标签**：每篇加 5-10 个相关标签
7. **发布时间**：晚 18-22 点流量高峰
8. **系列化**：同一主题出系列笔记

### 热门选题方向
- 🛠️ 开发工具推荐（VS Code 插件、AI 编程工具）
- 💼 程序员职场（面试技巧、跳槽经验、薪资）
- 📚 学习路线/资源推荐
- 🐛 Bug 解决经验（用「踩坑」包装）
- 💡 编程小技巧/冷知识
- 🏠 程序员生活（桌面分享、工位布置）

## 输出模板

```markdown
📌 {标题 + 1-2个emoji}

{开头2-3行：痛点 + 共鸣 + emoji}

——

1️⃣ {要点一}
{2-3行说明}
{配图建议}

2️⃣ {要点二}
{2-3行说明}

3️⃣ {要点三}
{2-3行说明}

——

✅ 总结：{一句话概括}

觉得有用的话 点赞收藏不迷路～
有问题评论区见👇

#程序员 #编程 #技术分享 #{技术关键词} #效率提升
```

### 封面图设计建议

提示用户封面图设计要点：
- 背景：代码编辑器截图 / 渐变色背景
- 大字标题：白色或亮色，粗体
- 装饰元素：emoji、图标
- 尺寸：3:4 竖版
- 工具：Canva、美图秀秀、醒图
